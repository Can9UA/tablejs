1) Название: Реализовать редактируемую таблицу без использования библиотек

      2) Краткое описание:
      - Вид таблицы, которую необходимо реализовать можно увидеть тут https://app.moqups.com/mr.vadim.ivanov@gmail.com/Jyx1sd3HWL/view
      - Таблица должна содержать три колонки с произвольным содержимым (желательно нумерованный список имен)
      - В конце каждой строчки должен быть чекбокс. Если он отмечен, то при нажатии на кнопку Delete вся строчка удаляется
      - По двойному клику на ячейках с данными появляется инпут для редактирования содержимого. Для подтверждения редакnирования можно добавить кнопку или менять содержимое при потере фокуса инпута
      - В заголовке колонок таблицы должны быть стрелки, которые позволяют сортировать таблицу (a-z, 0-9)

      3) Требования:
      - Поддержка: ие9+, последний хром, опера, сафари, фаерфокс
      - Библиотеки использовать запрещено, только чистый, нативный javascript
      - Готовое задание должно быть оформлено ссылкой на github и ссылкой на хостинг, чтобы можно было увидеть приложение вживую
      - Модульность, комментарии, ясное именование функций
      - Использование спецификации ES5
      - В качестве code styleguide используем этот свод правил https://github.com/airbnb/javascript (не обращаем внимание на то, что относится к ES6: let, const и т.д.)

      Бонусные требования: 
        1) реализовать фильтр.
        2) наполнять таблицу данными
